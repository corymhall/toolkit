# Archive

This directory preserves reference material from earlier workflow experiments.

Current contents:

- `gastown/` — the older Gastown-oriented workflow assets
- `gascity/` — the newer Gas City-oriented workflow assets

These directories are intentionally archived rather than deleted so ideas,
templates, and historical design decisions remain easy to inspect.

They are not part of the active workflow surface for this repo.
Prefer the root-level docs, agents, general skills, and `.config/wt.toml`
unless you are explicitly researching or reusing archived material.
