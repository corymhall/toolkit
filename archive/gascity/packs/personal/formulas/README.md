# Personal Pack Formula Candidates

The `personal` pack is the place for formulas that are intentionally more
autonomous than `work`, but still explicit.

Candidate direction:

- personal-only direct-commit or merge-friendly flows may belong here later
- they should not be treated as shared `base` formulas
- they should not be selectable from a `work` rig by accident

Open questions:

- Which personal-only landing behaviors do we actually want?
- Should the first personal-specific formula still stop short of merge, or do
  we want a fully local direct-commit path?
