# Gas City Namespace

This is the canonical home for the current Gas City model in this repo.

It contains:

- `formulas/` — current workflow formulas and expansions
- `skills/` — current Gas City-oriented skills
- `packs/` — local Gas City pack family
- `cities/` — local sample city assets
- `docs/templates/` — shared spec / plan templates used by the formulas

Planning docs do not live here. They stay under
`docs/plans/` at the repo root because they are session-planning artifacts for
this repo rather than installable namespace content.

Use this namespace for new convoy-first, graph-first, and pack-based work.
